# Security Policy

Neon Rain is an early-stage local visualizer and does not currently provide a network service.

Please do not publish exploit details in a public issue. Send a private message to the repository owner through GitHub with a concise description, affected revision, reproduction steps, and potential impact.

Security reports involving helper scripts, command execution, file parsing, dependency behavior, or unsafe handling of local media are especially relevant.

Only the latest commit on `main` is currently supported for security fixes.
