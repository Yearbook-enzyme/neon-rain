# Neon Rain v0.1.0-alpha.2

This portability-focused alpha makes Neon Rain available as both a reproducible Nix flake and a downloadable generic Linux archive.

## Highlights

- GPU-accelerated world-space Matrix rain with bloom and cinematic motion
- PipeWire music responsiveness and MPRIS media-player awareness
- Optional lyrics, moodbar, track-profile, and timeline enrichment
- Native Ubuntu build validation outside Nix
- Generic `x86_64` Linux tarball built on Ubuntu 22.04
- Dependency-checking launcher, desktop entry, and per-user installer
- Portable SHA-256 checksum
- New `--help` and `--version` commands
- Clean-profile and interactive bundle testing

## Run with Nix

```bash
nix run github:Yearbook-enzyme/neon-rain/v0.1.0-alpha.2
```

Press `Escape` to exit.

## Generic Linux archive

Download these two release assets:

- `neon-rain-0.1.0-alpha.2-x86_64-linux.tar.gz`
- `neon-rain-0.1.0-alpha.2-x86_64-linux.tar.gz.sha256`

Verify and extract:

```bash
sha256sum -c neon-rain-0.1.0-alpha.2-x86_64-linux.tar.gz.sha256
tar -xzf neon-rain-0.1.0-alpha.2-x86_64-linux.tar.gz
cd neon-rain-0.1.0-alpha.2-x86_64-linux
./neon-rain
```

Install for the current user from the extracted directory:

```bash
./install-user.sh
```

## Tested and verified

- NixOS, KDE Plasma Wayland
- AMD Radeon RX 580 using Vulkan through RADV
- Empty temporary home-directory profile
- Native Cargo compilation and tests on Ubuntu CI
- Generic Linux archive structure and checksum
- Generic bundle CLI, rendering, music response, F1 help, and F2 signal inspector

## Compatibility scope

The generic archive targets current 64-bit glibc-based Linux desktops. Host graphics drivers, Vulkan or OpenGL loaders, desktop-session libraries, and Fontconfig remain distribution-provided. Broader real-machine Linux testing is still needed.

Bug reports should include the distribution and version, desktop/session, GPU, graphics backend, launch method, reproduction steps, and sanitized terminal output.
