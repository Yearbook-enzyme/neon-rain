# Neon Rain v0.1.0-alpha.1

This is the first packaged alpha of Neon Rain: a living, music-reactive Matrix rain visualizer built in Rust with `wgpu`.

## Highlights

- GPU-accelerated world-space Matrix rain
- Persistent glyph streams, bloom, depth, and cinematic motion
- PipeWire-based music responsiveness
- MPRIS-aware player metadata support
- Optional lyrics, moodbar, profile, and timeline enrichment
- Local image-field coupling and apparition behavior
- Built-in help and signal-inspector overlays
- Reproducible Nix package and development shell
- Automated formatting, tests, Clippy, and Nix package checks

## Tested environment

- NixOS
- `x86_64-linux`
- KDE Plasma on Wayland
- AMD Radeon RX 580
- Vulkan through RADV

## Run

After the repository becomes public:

```bash
nix run github:Yearbook-enzyme/neon-rain/v0.1.0-alpha.1
```

For the current private checkout:

```bash
nix run .
```

Press `Escape` to exit.

## Alpha limitations

- Only the NixOS/Nix `x86_64-linux` path has been tested as a release target.
- Configuration and controls may change.
- Optional media enrichment depends on local programs and data.
- Real screenshots and demo footage are still planned.
- Other Linux distributions, Windows, macOS, and mobile are not yet packaged.

Bug reports should include the operating system, desktop/session, GPU, graphics backend, launch method, reproduction steps, and sanitized terminal output.
